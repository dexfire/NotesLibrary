用户登录

简要描述：

  • 用户登录接口

请求URL：

  • http://xx.com/api/user/login

请求方式：

  • POST

参数：

参数名 必选 类型 说明
username 是 string 用户名
password 是 string 密码

返回示例

  1.   {
  2.     "error_code": 0,
  3.     "data": {
  4.       "uid": "1",
  5.       "username": "12154545",
  6.       "name": "吴系挂",
  7.       "groupid": 2 ,
  8.       "reg_time": "1436864169",
  9.       "last_login_time": "0",
  10.     }
  11.   }

返回参数说明

参数名 类型 说明
groupid int 用户组id，1：超级管理员；2：普通用户
