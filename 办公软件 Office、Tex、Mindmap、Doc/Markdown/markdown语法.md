# MarkDown 语法

##语法简明概述
1.分段 ```2个回车```
2.换行 ```2个空格 + 回车```
3.标题 ```# ~ ######```，几个```#```表示几级标题，最多六级
4.强调 ```**加粗**，`高亮`，__强调__，_强调_，*强调*，~~删除~~```
5.引用 ```> 文本```注意```>```后面有个空格
6.表格 ```使用 - | 分割行和列，: 用于对齐控制```
7.代码 ```四个空格```或者，``` '```'代码'```' ```
## 特殊用法
1.转义 <code>`</code><code>`</code>
2.HTML 直接使用html语法即可，```<tag>Content</tag>```
